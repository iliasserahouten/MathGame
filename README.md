# MathGame

MathGame est un jeu éducatif en Python conçu pour aider les utilisateurs à améliorer leurs compétences en mathématiques de manière ludique et interactive. Le jeu propose différents types d’équations et calculs, permettant aux utilisateurs de tester leurs connaissances tout en s’amusant.

## Fonctionnalités

- **Connexion et Inscription** : Créez un compte ou connectez-vous avec votre ID utilisateur.
- **Menu Joueur** : Accédez au jeu, consultez le classement et gérez votre session.
- **Modes de Jeu** :
  - Équation Quadratique
  - Équation Linéaire
  - Calculs Simples
- **Classement** : Consultez les 10 meilleurs scores et votre score personnel.
- **Gestion des Utilisateurs et des Scores** : Les scores sont enregistrés dans un fichier texte sécurisé.
- **Interface Graphique Interactive** : Grâce à Tkinter et MessageBox pour des notifications claires.

## Bibliothèques utilisées

- **Tkinter** : Création de l’interface graphique.
- **Hashlib** : Sécurisation des mots de passe.
- **Math** : Fonctions mathématiques pour les calculs.
- **MessageBox (Tkinter)** : Affichage de messages et alertes.
- **Numpy** : Calculs numériques et manipulation de tableaux.
